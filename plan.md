# 三国霸业游戏开发计划

## 项目概述
使用 Python + Pygame 开发一款画面精美的三国霸业策略游戏，支持简体中文界面，操作方式与经典三国霸业游戏一致。

## 技术栈
- **语言**: Python 3.10+
- **游戏框架**: Pygame 2.5+
- **UI 组件**: 自定义 UI 组件（仿古典风格）
- **数据存储**: JSON（配置、存档）
- **日志**: Python logging 模块

## 数据结构

### 武将数据
```python
{
    "id": int,
    "name": str,           # 姓名
    "courtesy_name": str,  # 字
    "faction": int,        # 所属势力
    "location": int,       # 所在城池
    "attributes": {
        "war": int,        # 武力
        "int": int,        # 智力
        "pol": int,        # 政治
        "chr": int,        # 魅力
    },
    "skills": [],          # 技能列表
    "loyalty": int,        # 忠诚度
    "troops": int,         # 带兵数
    "status": str          # 状态（空闲、出征、任务等）
}
```

### 城池数据
```python
{
    "id": int,
    "name": str,           # 城池名
    "faction": int,        # 所属势力
    "population": int,     # 人口
    "agriculture": int,    # 农业
    "commerce": int,       # 商业
    "defense": int,        # 城防
    "governor": int,       # 太守
    "officers": [],        # 城中武将列表
    "troops": int,         # 兵力
    "resources": {
        "gold": int,       # 金钱
        "food": int        # 粮草
    }
}
```
